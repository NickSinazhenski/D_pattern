# Prototype + Decorator + Command (JS)

Мини-проект демонстрирует три паттерна:
- Prototype: создание копий документов через прототипы.
- Decorator: добавление водяной метки.
 - Command: выполнение операций над документами через команды.

## Структура
- `src/documents.js` — документы, прототипы и декоратор водяной метки.
- `src/document_commands.js` — команды для операций над документами.
- `index.js` — пример сценария создания копии и добавления метки.

## Запуск

```bash
node index.js
```

Вы увидите создание и клонирование документа, применение водяной метки через команды.
