# Composite + Strategy (JS)

Мини-проект демонстрирует два паттерна:
- Composite: задачи объединяются в дерево (проекты, подзадачи, группы).
- Strategy: каждая задача выполняется по выбранной стратегии (Agile, Waterfall, Spiral, RUP).

## Структура
- `src/tasks.js` — базовые классы задач и композитов.
- `src/strategies.js` — набор стратегий выполнения.
- `index.js` — пример сборки дерева задач и запуска разных стратегий.

## Запуск

```bash
node index.js
```

Вы увидите несколько сценариев выполнения одной и той же структуры задач, но с разными алгоритмами.
